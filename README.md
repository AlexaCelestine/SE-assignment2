# SE-Assignment2

## Project Description
This is a simple C++ project demonstrating basic Git operations, branching, pull requests, and merge conflicts. It includes a Hello World program with added features.

## Installation Instructions
1. Clone the repository:
   git clone https://github.com/AlexaCelestine/SE-assignment2.git 
2. Navigate to the project folder:
   cd SE-assignment2
3. Compile the C++ program (requires g++):
   g++ hello.cpp -o hello
4. Run the program:
   ./hello # on Windows: hello.exe

## Usage
- The program prints greetings to the console.
- Demonstrates adding features on separate branches and resolving conflicts.
